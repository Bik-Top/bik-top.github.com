Функции API

API LiqPAY делает доступными для разработчиков функции платформы LiqPAY. На данный момент открыты следующие возможности:

  • а) API для перевода средств и контроля транзакций
  • б) по приёму платежей в пользу мерчанта
  • в) API Open Billing для создания собственной платежной системы
  • г) API для банков и финансовых организаций

Регистрация в API

Для получения данных мерчанта необходимо пройти регистрацию мерчанта Реквизиты мерчанта (ID и пароль).

Если Вы еще не стали мерчантом, Вас автоматически переведет на страницу регистрации.
